## Step 2 — Announce before every tool (speak, then call)

Before **each** tool call, say one short line out loud, then call the tool:

| Tool | Say something like |
|------|-------------------|
| `cursor_ask` | "I'll ask Cursor — about a minute." |
| `cursor_submit` | "Sending that to Cursor to work on." |
| `cursor_status` | "Checking what Cursor is doing." |
| `cursor_recall_answer` | "Pulling up Cursor's last answer." |

You must say these lines yourself — the server does not speak them for you.
